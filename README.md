# Risk Management Project

This repository contains a JupyterLab workflow for the course project “Valuation and Hedging – Options”.

## Contents

Files:
- `ASML_data.csv`  
  Daily ASML market data used to compute spot price, strike mean, and historical volatility.
- `ECB Data Portal_20260302103342.csv`  
  ECB yield curve data used to extract the 1-year spot rate series as the risk-free rate.
- `risk_manage.ipynb` 
  Main notebook containing the full solution with code + markdown write-up.


## Requirements

Packages:
- pandas
- numpy
